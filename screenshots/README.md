# Screenshots

This folder will contain visual documentation:

- iReport Designer interface
- Report layout design
- Putty deployment process
- Finacle FINRPTC menu
- FINRPT testing
- Final report output

## To Add Screenshots

1. Take screenshots of each step
2. Save with descriptive names (e.g., `01_ireport_wizard.png`)
3. Upload to this folder
4. Update main README with image links

## Screenshot Guidelines

- Use high resolution (at least 1920x1080)
- Highlight important areas
- Remove sensitive information
- Use PNG format for clarity
